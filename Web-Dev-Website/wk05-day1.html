Tiling Background Images

tiling a background image horizontally and centered.

background-image:    url("../images/vector.jpg");
background-repeat:   repeat-x;
background-position: center;

Background Linear Gradient

Using special values with the background-image property, users can crate a variety of gradient backgrounds.

background-image:
  linear-gradient(60deg,red,white 30%,blue);

Multiple Backgrounds

Separate bg images and their controls with commas.

background-image:
	url("../images/penguin-icon_.png"),
	url("../images/desert.jpg");

background-size: 150px, 400px;
background-repeat: no-repeat, repeat;
background-position: 0% 120%, center center;

Cover Background

A developer can set a full background image with background size set to cover

background-image: url("../images/snowycity.jpg");
background-position: center;
background-repeat: no-repeat;
background-size: cover;
