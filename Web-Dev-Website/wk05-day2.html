Basic Image Types

JPG

  • Support 16.7 Million Colors
  • Lossy Compression
  • Small File Sizes
  • Not A Good Working File Format
  • No Animation, No Transparency
  • Go To
  • Great For Graidents

GIF

  • Supports Only 256 Colors
  • Lossless Compression
  • Makes Sorta Small File Sizes
  • Good For Simple Line Art (Cartoons)
  • 1 Of The 256 Colors Can Be Transparent
  • Supports Animation

PNG

  • Good For Line Art / Cartoons
  • Supports 16.7 Million Colors
  • Uses A Lossless Compression
  • Sadly, No Animation
  • But, Supports Animation AND Alfa Fade
  • Filesize Not As Good As JPG
